# CSLA Steering Committee Meeting - December 19, 2025

## Context

First steering committee meeting for the CSLA retail lending integration. This meeting covered the pension migration progress (moving from direct Citi integration to CSLA-intermediated) and set the foundation for the retail lending project.

## Key Topics Covered

### 1. Pension Migration to CSLA

The existing pension securities lending was being migrated from a direct Nordnet↔Citi setup to go through CSLA (Sharegain). This was the first step before building the retail solution.

### 2. Retail Program Planning

Initial discussion of the retail lending program scope, timeline, and integration approach. The retail solution would be built as a new NNX cloud service.

### 3. Integration Architecture

Review of how the tri-party model (Nordnet / CSLA / Citi) works and the data flows between parties.

## Notes

Full details from the PDF presentation were reviewed. Key architectural decisions and program structure were established at this meeting, which then led to the detailed April 2026 workshop (see csla-workshop-april-2026.md).
